行内块元素：

      特点：
          1、和其他元素都在一行显示【不独占一行】
          2、高、行高以及顶部和底部的边距可修改的
          3、宽度就是他文字的宽度、可修改的


   功能：用户加强内容显示，控制细节 调试




   元素：
       input：定义输入控件

           type="text"                    输入框：默认
           type="password"              密码
           type='checkbox'              多选按钮
           type="radio"                 单选按钮
           type='button' value='登录'>   按钮
           type='submit' value='登录'>   提交           
           type='file'                  选取文件
           type='reset'                 重置

        textarea：文本域   （文字的自由调节框）
        select  ：定义选择列表（配合option来使用）（下拉选择框）
        img     ：定义图像

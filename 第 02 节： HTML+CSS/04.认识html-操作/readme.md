HyperText Markup Language : 超文本标记语言


二、点html什么意思？

	.html    是一个扩展名    浏览器   解析或播放


	.mp4    ====>播放器     解析  或播放
	.mp3
	.exe
	.jpg
	.png

	.html ===》浏览器

三、这是什么玩意

	<!DOCTYPE html> ===》声明帮助浏览器正确地显示网页

	<html>   ===》根
		<head> ===》头
		</head>

		<body>  ===》身体
		</body>
	</html>
	
四、标记语言

	<英文单词或字母组合>

	<body>  开始
	</body> 结束

五、标记属性

	<英文单词或字母组合 style class id ...>

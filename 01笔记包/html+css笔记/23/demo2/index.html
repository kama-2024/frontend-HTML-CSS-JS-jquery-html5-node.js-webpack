2000年，出演了《黑洞》饰演三陪女，马伊琍的表演事业得到了更好的提升，在电视剧《前世今生》中饰演美貌且气质超凡的秦婉莹。2001年，出演电视剧《白天睡觉的女人》饰演李秀 [10] ，在电影《西施眼》中饰演施雨，该剧在2002年美国夏威夷国际电影节中获评审团大将及亚洲最佳电影。 [10] 2002年，因出演电视剧《还珠格格Ⅲ之天上人间》中夏紫薇一角而被大众所熟知。 2004年，出演了由黄磊指导的《天一生水》饰演一个混血姑娘明妮。2005年，出演张黎导演的电视剧《锦衣卫》 [11] 在剧中就出演了一个这样的倾国美人客印月。2006年，参演《乔家大院》扮演的“江雪瑛”。2007年，出演由赵宝刚指导的青春励志电视剧《奋斗》夏琳一角而知名度得到进一步的提升。 2008年，因拍电视剧《为谁心动》时无悔剃光了头，凭借影片《江北好人》，马伊琍战胜了李冰冰、范冰冰两位大热候选人，成为了长春电影节影后。 [12-13] 2009年，与佟大为共同出演了电视剧《婚姻保卫战》演出夫妻间事。 [14] 同年马伊琍出演了首部反映豫商故事的电视剧《经纬天地》在CCTV-8播出，这也是她继《乔家大院》之后首次登，因拍电视剧《为谁心动》时无悔剃光了头，凭借影片《江北好人》，马伊琍战胜了李冰冰、范冰冰两位大热候选人，成为了长春电影节影后。 [12-13] 2009年，与佟大为共同出演了电视剧《婚姻保卫战》演出夫妻间事。 [14] 同年马伊琍出演了首部反映豫商故事的电视剧《经纬天地》在CCTV-8播出，这也是她继《乔家大院》之后首次登，/>因拍电视剧《为谁心动》时无悔剃光了头，凭借影片《江北好人》，马伊琍战胜了李冰冰、范冰冰两位大热候选人，成为了长春电影节影后。 [12-13] 2009年，与佟大为共同出演了电视剧《婚姻保卫战》演出夫妻间事。 [14] 同年马伊琍出演了首部反映豫商故事的电视剧《经纬天地》在CCTV-8播出，这也是她继《乔家大院》之后首次登，/>因拍电视剧《为谁心动》时无悔剃光了头，凭借影片《江北好人》，马伊琍战胜了李冰冰、范冰冰两位大热候选人，成为了长春电影节影后。 [12-13] 2009年，与佟大为共同出演了电视剧《婚姻保卫战》演出夫妻间事。 [14] 同年马伊琍出演了首部反映豫商故事的电视剧《经纬天地》在CCTV-8播出，这也是她继《乔家大院》之后首次登，/>因拍电视剧《为谁心动》时无悔剃光了头，凭借影片《江北好人》，马伊琍战胜了李冰冰、范冰冰两位大热候选人，成为了长春电影节影后。 [12-13] 2009年，与佟大为共同出演了电视剧《婚姻保卫战》演出夫妻间事。 [14] 同年马伊琍出演了首部反映豫商故事的电视剧《经纬天地》在CCTV-8播出，这也是她继《乔家大院》之后首次登

演艺经历

15岁时参加上海电视台的综艺舞蹈队，正式自食大学期间的旧照大学期间的旧照(6张)其力自己赚取学费。 [9] 1993年，被选中主演电影《刘海粟》，那是她的第一次触电经历，收获800元片酬。1994年，考入上海戏剧学院94级表演系本科。 [9] 1996年，在电视剧《真空爱情记录》中饰演夏文心，讲述了一群大学毕业生为了工作和爱情奋斗的故事。

个人生活

2008年，马伊琍和她小8岁的文章结婚，婚后二人育有两个女儿。

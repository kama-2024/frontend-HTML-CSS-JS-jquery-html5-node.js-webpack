CSS选择器-元素选择器  （元素选择器）


   有哪些：

           *           通配选择器       选择所有的元素
                       使用场景：       *{margin:0;padding:0;}
                                         外边距      内边距

           element     元素选择器       选择指定的元素


           #id         id选择器           选择元素id属性值为#id的元素                     领导

           .class      class选择器     选择元素class属性值为.class的元素              员工
